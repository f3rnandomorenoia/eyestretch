Cómo usarlo
  1. Abre esta página en el iPhone.
  2. Pulsa Desbloquear audio.
  3. Pulsa Tono Web Audio (fuerte).
  4. Pulsa Tono HTML audio.
  5. Si suena el tono HTML pero no el de Web Audio, el problema es específico de Web Audio.
  6. Si no suena ninguno, entonces el iPhone no está sacando audio en Safari o el volumen físico/media está mal.
  7. Si el estado queda en running y el workaround en allowed, el fix técnico está activado.
Interpretación rápida
  • running + allowed + suena Web Audio → arreglado.
  • running + allowed + suena HTML pero no Web Audio → sigue habiendo un problema en la ruta Web Audio.
  • running + allowed + no suena ninguno → problema externo al fix (volumen/salida/sistema/Safari).
